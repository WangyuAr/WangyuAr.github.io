每天一道leetcode322这题dfs居然超时

给定不同面额的硬币 coins 和一个总金额 amount。编写一个函数来计算可以凑成总金额所需的最少的硬币个数。如果没有任何一种硬币组合能组成总金额，返回 -1。

输入: coins = [1, 2, 5], amount = 11
输出: 3 
解释: 11 = 5 + 5 + 1

输入: coins = [2], amount = 3
输出: -1

思路： 首先想到的肯定是递归回溯深搜啊，可惜超时了。超时代码：

class Solution {
    int[] coins;
    int amount;
    int res = 10000;
    public int coinChange(int[] coins, int amount) {
        if(amount==0)
            return 0;
        this.coins = coins;
        this.amount = amount;
        for(int i =0;i<coins.length;i++){
            if(coins[i]>amount)
                continue;
            dfs(coins[i],0);
        }
        if(res==10000)
            return -1;
        else
            return res+1;
    }
    public void dfs(int curAmount, int curNum) {
        if(curAmount>amount){
            return;
        }
        if(curAmount == amount){
            if(curNum<res)
                res = curNum;
            return ;
        }
        for(int i =0;i<coins.length;i++){
            if(coins[i]>amount)
                continue;
            dfs(curAmount+coins[i], curNum+1);
        }
    }
}
